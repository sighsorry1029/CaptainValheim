# Changelog

## 1.0.0

- Initial release of the CaptainValheim shield combat module.
- Added shield primary attacks, shield throws, shield charges, projectile reflection, and block-charge support.
- Added stat-derived shield scaling for damage, push, stamina, durability, target count, travel distance, speed, and cooldowns.
- Added synced `CaptainValheim.yml` configuration with global defaults and per-shield prefab overrides.
- Added WarfareTweaks bridge support for shield hit context recognition.
